**Movie App**
A React-based movie application to explore trending movies and search functionality.

**Features**
- Search movies
- Trending movies
- Fast UI with Vite
- Styled with Tailwind CSS

**Tech Stack**
- React
- Vite
- Tailwind CSS
- Appwrite



**Run Locally**
git clone https://github.com/bhavikas-9999/movie-app.git
cd movie-app
npm install
npm run dev

**Setup Required**
- Add `http://localhost:5173` in Appwrite Platforms
